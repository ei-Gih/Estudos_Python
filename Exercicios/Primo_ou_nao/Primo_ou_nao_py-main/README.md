# Primo_ou_nao_py

Verifica se um numero é Primo, Programa feito em Python; Recebe um numero; Conta quantos divisores ele tem (de 1 até ele);Se divisores == 2 → primo

## Lógica completa:

1. while True → repete o programa
2. try → proteger a entrada do usuário (caso ele digite letra)
3. input → pede um número
4. função verificar_primo → diz se é primo
5. print → exibe o resultado
6. input → pergunta se quer continuar
7. if != "s" → break
